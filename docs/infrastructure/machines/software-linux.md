# Software: Linux

Every user Linux machines comes equipped with...

- Standard userland utilities: `mv`, `cp`, `echo`, `chmod`, `man` and so on.
- [`rg`](https://github.com/BurntSushi/ripgrep), a faster grep.
- [`plocate`](https://plocate.sesse.net/), a universal file-finder
- [`btop`]
- [`nmap`] / [`netcat`]
- [`curl`] / [`wget`]
- [`vim`] / [`neovim`] / [`nano`]
- [`exa`]
- [`make`] + [`cmake`] 
- [`clang`](https://clang.llvm.org/)
- [`lldb`]() / [`gdb`](), debugging utilities
- [`rustup`](https://rustup.rs/) + [`cargo`](https://doc.rust-lang.org/cargo/), the basic rust toolkit
- [`valgrind`]
- [`python3`](https://www.python.org/), objectively the best scripting language
- `bash`, a good shell
- [`nnn`](https://github.com/jarun/nnn), a nice file manager
- [`tmux`](https://github.com/tmux/tmux/wiki), the classical terminal multiplexor
- [`libreoffice`](https://www.libreoffice.org/), the libre office suit
- [`LibreWolf`](https://librewolf.net/), a very privacy-aware browser.
- [`ffmpeg`]
- [`bmon`]
- [`jq`]
- [`perf`]
- [`tcpdump`]
- [`lynis`], the automatic security auditing tool.
- [`croc`]
- ['ethtool`]
- [`wakeonlan`]
